Core Language

Cryptography

Evasion

Infrastructure Core

Runtime Technology

Fileless Memfd Loader

The agent never touches the disk. It utilizes the memfd_create syscall to load and execute the binary directly from system memory, providing a zero-trace forensic footprint.

Network Security

L7 TLS Mimicry

All communication is wrapped in an encrypted tunnel that mimics standard HTTPS browser fingerprints (Chrome/Firefox), evading deep packet inspection (DPI) and traffic analysis.
